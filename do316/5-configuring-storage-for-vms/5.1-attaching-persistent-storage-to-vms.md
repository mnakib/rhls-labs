

This example shows how to connect virtual machines to different persistent storage services based on performance and cost needs for the running applications.


1 - Create the VMs

Create two VMs named db-server and web-server


2- Attach fast storage the db-server server

Prepare a **5 GiB** blank disk for the **db-server** database server. Attach the disk to the VM by using a _virtio_ interface. Use the _ocs-external-storagecluster-ceph-rbd-​virtualization_ storage class, which provides fast block storage.

3- Attach mid-range storage the web-server server

Prepare a **10 GiB** blank disk for the **web-server** web server. Attach the disk to the VM by using a _virtio_ interface. Use the _nfs-storage_ storage class.


Check the disks have been correctly attached

```
# virtctl console db-server | web-server
```

```
# lsblk
```
