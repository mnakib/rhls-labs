# Configuring a Multihomed Virtual Machine


### Create a working project

```bash
oc new-project multihomed-demo
```

### Create a NetworkAttachedDefintion

Create a NAD with below config:

Name: br0-network
Network Type: OVN Kubernetes secondary localnet network
Bridge Mapping: br0-network


### Create a multihomed VM named `multihomed-vm`

While creating the VM, customize it to add a second NIC with below config

  __Name:__ external-interface
  __Model:__ virtio
  __Network:__ network-multihomed/br0-network
  __MAC address:__ 00:00:00:00:00:01

Assign the 00:00:00:00:00:01 MAC address to the interface. The DHCP server that is running on the utility machine is configured to assign the 192.168.51.15 IP address to that MAC address.



### Check the VM IP multihomed config

Get the VM credentials from the Console tab

Connect to the VM console

```bash
virtctl console multihomed-vm
```

Check that the VM has two NICs

```bash
ip a
```
