# Configuring External Access to Virtual Machines

### Create a new working project

```bash
# Create a new working project named `externalaccess-demo`
oc new-project externalaccess-demo

# Swith the `externalaccess-demo` project
oc project externalaccess-demo
```

### Create a Web server VM from the `rhel10-helloworld` boot volume

Create a VM named `web-vm` from the `rhel10-helloworld` boot volume in the `externalaccess-demo` project



### Expose the VM through a service

```bash
virtctl expose vm web-vm --name=web-vm-svc \
  --type=ClusterIP --port=80 --target-port=80
```

### Create a route to expose the service

```bash
oc expose service web-vm-svc
```
